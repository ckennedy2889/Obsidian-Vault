# Bootstrapping

In Fixed Income, the process of deriving [[Spot rate|spot rates]] or zero-coupon rates from the [[Par curve]], starting with the shortest maturity and using each solved rate as an input for the next maturity.

For the full Level II explanation, see [[Bootstrapping Spot Rates]].

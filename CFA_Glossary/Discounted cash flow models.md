# Discounted cash flow models

**Program:** CFA® Program

## Definition (Level III)

(DCF models) Valuation models that express the idea that an asset’s value is the present value of its (expected) cash flows.

## Definition (Level I)

Valuation models that estimate the intrinsic value of a security as the present value of the future benefits expected to be received from the security.

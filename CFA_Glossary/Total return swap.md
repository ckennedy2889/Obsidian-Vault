# Total return swap

**Program:** CFA® Program

## Definition (Level III)

A swap in which one party agrees to pay the total return on a security. Often used as a credit derivative, in which the underlying is a bond.

## Definition (Level I)

A swap in which one party agrees to pay the total return on a security. Often used as a credit derivative, in which the underlying is a bond.

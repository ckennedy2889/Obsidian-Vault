# Indexing

**Program:** CFA® Program

## Definition (Level III)

A common passive approach to investing that involves holding a portfolio of securities designed to replicate the returns on a specified index of securities.

## Definition (Level I)

An investment strategy in which an investor constructs a portfolio to mirror the performance of a specified index.

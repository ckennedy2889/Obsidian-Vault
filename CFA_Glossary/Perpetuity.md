# Perpetuity

**Program:** CFA® Program

## Definition (Level II)

A perpetual annuity, or a set of never-ending level sequential cash flows, with the first cash flow occurring one period from now.

## Definition (Level I)

A perpetual annuity, or a set of never-ending level sequential cash flows, with the first cash flow occurring one period from now. A bond that does not mature.

# Tactical asset allocation

**Program:** CFA® Program

## Definition (Level III)

Asset allocation that involves making short-term adjustments to asset class weights based on short-term predictions of relative performance among asset classes.

## Definition (Level I)

The decision to deliberately deviate from the strategic asset allocation in an attempt to add value based on forecasts of the near-term relative performance of asset classes.

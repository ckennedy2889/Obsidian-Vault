# Dividend discount model

**Program:** CFA® Program

## Definition (Level I)

(DDM) A present value model that estimates the intrinsic value of an equity share based on the present value of its expected future dividends.

## Definition (Level II)

(DDM) A present value model of stock value that views the intrinsic value of a stock as present value of the stock’s expected future dividends.

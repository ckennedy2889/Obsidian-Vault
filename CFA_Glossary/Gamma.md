# Gamma

**Program:** CFA® Program

## Definition (Level III)

A numerical measure of the sensitivity of delta to a change in the underlying’s value.

## Definition (Level I)

A numerical measure of how sensitive an option’s delta (the sensitivity of the derivative’s price) is to a change in the value of the underlying.

## Definition (Level II)

A measure of how sensitive an option’s delta is to a change in the underlying. The change in a given instrument’s delta for a given small change in the underlying’s value, holding everything else constant.

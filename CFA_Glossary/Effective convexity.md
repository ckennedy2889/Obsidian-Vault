# Effective convexity

**Program:** CFA® Program

## Definition (Level III)

A second-order effect that describes how a bond’s interest rate sensitivity changes with changes in yield. Effective convexity is used when the bond has cash flows that change when yields change (as in the case of callable bonds or mortgage-backed securities).

## Definition (Level I)

A curve convexity statistic that measures the secondary effect of a change in a benchmark yield curve on a bond’s price.

## Definition (Level II)

Sensitivity of duration to changes in interest rates.
